웹 접근성

웹 접근성을 생각하며 시멘틱 태그를 사용하였으며 clip class를 사용하여 구간,아이콘등을 설명하는 텍스트를 넣어 스크린 리더기가 읽을 수 있게 코딩을 했습니다.

반응형 디자인

웹 페이지 전체 사이즈 수정에 대비하여 rem단위를 이용하여 만들었으며, Mobile First 방식으로 마크업 하였습니다.

사용자 경험을 생각하며 디자인

가사창을 모달로 만들어 왼쪽에 숨겨두고 펼쳐 볼수있도록 하여 스크롤을 최소화 시켰습니다.

회원가입, 로그인 폼 구현

정규식을 사용하여 회원가입 폼을, localstorage와 sessionstorage를 활용하여 val를 받아 로그인이 됐을때의 모습도 구현 연습을 해봤습니다.
